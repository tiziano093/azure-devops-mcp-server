# Code of Conduct

## Our Standard

This project adopts the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

We expect contributors and maintainers to keep discussion technical, respectful, and focused on improving the project.

## Expected Behavior

- Be respectful in technical disagreement.
- Assume good intent, but challenge ideas rigorously.
- Keep bug reports, reviews, and proposals concrete and reproducible.
- Avoid harassment, personal attacks, discrimination, and disruptive behavior.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, review threads, and other public communication channels related to this repository.

## Reporting

To report unacceptable behavior, contact the maintainer privately through GitHub or by the security contact defined in [SECURITY.md](SECURITY.md) when appropriate.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with it.
